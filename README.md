# Project Name
Binary Star System Simulator

Developer Names:
Xinlu Yan

Date of project start:
January 2026

This project is a research-oriented software project developed using the
[Drasil](https://github.com/JacquesCarette/Drasil)
framework to systematically capture domain knowledge and automatically generate
software artifacts. The project models the time-dependent orbital evolution of a
binary star system from an initial physical configuration.

The folders and files for this project are as follows:

docs - Documentation for the project
refs - Reference material used for the project, including papers
src - Source code
test - Test cases
etc.
